🎲 ランダム数値生成の詳しい解説

Math.floor(Math.random() * (350 - 250 + 1)) + 250

ステップ1: Math.random()

Math.random() は 0以上1未満の小数をランダムに生成します。

例: 0.0, 0.123456, 0.789012, 0.999999... など
※ 1.0 は含まれません

ステップ2: (350 - 250 + 1) = 101

範囲の計算をします：

最大値 - 最小値 + 1 = 350 - 250 + 1 = 101
※ +1 をする理由：250, 251, 252... 350 の個数は101個だから

ステップ3: Math.random() * 101

0以上1未満の値に101を掛けると、0以上101未満の値になります。

Math.random() = 0.5 の場合
0.5 * 101 = 50.5

ステップ4: Math.floor() で切り捨て

小数点以下を切り捨てて整数にします。

Math.floor(50.5) = 50
Math.floor(100.9) = 100
取得可能な値: 0, 1, 2, 3, ... 100 (101個)

ステップ5: + 250 でオフセット

最小値(250)を足して、目標の範囲にシフトします。

0 + 250 = 250 (最小値)
100 + 250 = 350 (最大値)
取得可能な値: 250, 251, 252, ... 350

📝 一般的な公式

Math.floor(Math.random() * (最大値 - 最小値 + 1)) + 最小値

この公式で任意の範囲の整数をランダムに生成できます！
